# Gen N Ver – Biblioteca Python para Geração de Dados Fictícios

Biblioteca simples e leve desenvolvida para fornecer funções práticas para **geração de nomes, e-mails, usernames e senhas**, além de **validadores comuns** usados em aplicações reais.

Esta biblioteca não usa dependências externas e foi projetada para ser clara, didática e facilmente instalável via TestPyPI.

---

## Objetivo do Projeto

O propósito da biblioteca **Gen N Ver** é facilitar a criação de dados fictícios, podendo ser usada em:

- Testes de software  
- Protótipos e simulações  
- Exemplos de aula  
- Geração de dados para preenchimento automático  
- Situações onde nomes, emails ou senhas falsas são úteis  

Além disso, inclui validadores simples para:

- E-mail  
- Nome de usuário  
- Senha  

---

pip install -i https://test.pypi.org/simple/gen-n-ver
